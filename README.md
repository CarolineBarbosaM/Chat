# Chat com HTML e NODE.JS
Chat de conversas online.
## Executando o Projeto

### 1 - Clone o reposotório

```bash
git clone https://github.com/CarolineBarbosaM/Reset-API.git
```

### 2 - Instale a Dependências:

```bash
yarn add
```

### 3 - Execute o projeto

```bash
node serve.js
```
### 4 - Acesse o projeto

```bash
localhost:3000
```
